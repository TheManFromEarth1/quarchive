Archives of "{{bookmark.title}}"

Sometimes, if a webpage you bookmarked is offline, no longer working or has changed you can find a workable copy in some web archives.

The below links try to get a archival copy as close to your bookmark's created timestamp as possible ({{bookmark.created|datetimeformat(format="long")}}) - except where otherwise noted).

  • The Wayback Machine
  • archive.today
  • Google's cache (whatever they currently hold)

Why try multiple archives?

They all have different collection and retention policies.

The Wayback Machine, run by the Internet Archive, run a broad web crawl, picking up a lot, but sometimes remove pages.

archive.today (previously called archive.is) only store what people have requested but then only remove archived pages occasionally.

Google's cache can sometimes be useful if a page has gone offline shortly after being put up - before the others have archived it
