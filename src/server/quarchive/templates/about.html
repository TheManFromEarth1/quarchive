About Quarchive

Quarchive is a bookmarking site, showing metadata around your browser bookmarks

Full text search

Quarchive crawls and indexes your all your bookmarks, allowing you to search your bookmarks - even when the term you're searching for doesn't appear in the title.

Completely private by default

All bookmarks are completely private by default but there are ways to share your view of a bookmark via a magic link if you want.

See discussions

Quarchive shows reddit threads referring to your bookmarks, so you can more easily find people talking about things you are interested in.

Add bookmarks from mobile browsers

As Quarchive is sync-based and hooks into the built-in browser bookmarks subsystem it will automatically pick up any bookmarks made in a mobile browser (so long as you have browser-level sync enabled).

Backlinks

Some of the most interesting metadata is links between bookmarks. Quarchive shows these, so you can more easily find related pages. Helpful for finding where you found something from.

Free and Open Source

The source code for Quarchive (both the server and browser extension) is freely available on Github under the terms of the AGPL.

Quarchive is free to use but currently in beta - so beware of bugs!
